My top linux distros

Because Linux, the kernel and applications are opensource, there are many versions, or "distributions” of Linux and so these are my favorite ones and ones would recommend for either a beginner to the Linux operating system or someone willing to switch a distribution or even a pro!

  1. Debian

    Its stable, like most of the applications have a rare chance of crashing or there are less bugs because most things get fixed though security updates. And it is what ubuntu is based off. I’ve used Debian on 2 old machines that are 10 years old and they feel more new than on windows 10. The downside is that drivers(the things that makes the computer talk to the hardware) are a pain to install, especially the Wi-Fi drivers or that Debian stable, what makes Debian so stable, ends up with a bunch of stale software that isn’t old to be obsolete but isn’t the newest software.

  2. OpenSUSE Tumbleweed

    If a current windows user is a developer and needs the newest applications and the latest and greatest software. OpenSUSE is a great option as it’s very snappy and comes out with the newest applications like Arch Linux® (another distribution of Linux® that’s more in tuned to Sysadmins, and “Power Users”) but isn’t as unstable as Arch Linux®. Plus, it integrates the KDE plasma Desktop environment-the look and feel of a distribution that comes with a set up Application and is one of the first to get KDE features. Plus there’s a tool called YAst that acts like the windows control panel so windows users can get more intune with linux when switching from windows.

  3. Fedora

    A bit more stable than openSUSE Tumbleweed but is still a good distribution because the fedora development team are the ones who introduce new innovative standards to Linux® like having one of the first ones to integrate systemd when systemd-an init system, came out. Or Wayland, a new default display manager -a server that displays the actual windows of an application, as the default display manager. And its backed and used by Redhat, a business enterprise company that makes Redhat Linux® and gives support to Redhat Linux®. It doesn’t beat openSUSE because I Find Fedora Linux® full of software that is unnecessary for my needs, and it is limited to as what you can remove.

  4. Linux Mint

    Linux mint is a good option as it was my second distribution of Linux® and I used it on some old hardware, they don’t have snaps packages as the default when downloading software but it’s based on Ubuntu and so it depends on Ubuntu servers, it’s a great alternative to Ubuntu because while Ubuntu is a good distribution, Linux® mint is a good alternative and i recomend it.

  5. Kubuntu

    Kubuntu is a Ubuntu fork that uses the KDE plasma desktop environment as a desktop environment for Ubuntu and Ubuntu uses the gnome desktop environment by default. It was the first distribution that I used in Linux®, the one that made me like Linux® but it is basically Ubuntu but with KDE and after learning about Linux® from classes and from google and friends. That in my opinion there are better alternatives than to Ubuntu.

Arduino Music application

The "Arduino Music application" project is an Arduino that sings two old rock songs that are part of my jukebox application final for my C# class.
