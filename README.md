# Matts Development Blog/Personal Webiste

### Description
Matthew's Development Blog is a simple, static personal website that shares who I am, my hobbies, and my development journey.  
It includes an **About** section (background, goals, and interests), a **Projects** page (completed work and Linux dotfiles), and supporting pages like **Contact** and **FAQs**. The homepage serves as a landing page to highlight updates and recent work.

### Prerequisites
- Any modern web browser (Chrome, Firefox, Edge, Safari)
- To view locally: open `Final Project/index.html`
